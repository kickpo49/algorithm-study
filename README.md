> 백준허브를 활용하여 알고리즘 문제 풀이를 자동으로 기록하는 저장소입니다.

## 📌 소개

이 저장소는 **백준허브(BaekjoonHub)** 크롬 확장 프로그램을 사용하여 다양한 알고리즘 문제 풀이 플랫폼에서 해결한 문제들을 자동으로 GitHub에 저장합니다.

## 🎯 지원 플랫폼

백준허브는 다음과 같은 주요 코딩 테스트 플랫폼을 지원합니다:

| 플랫폼 | 설명 | 링크 |
|--------|------|------|
| 🥇 **백준 온라인 저지** | 대한민국 대표 알고리즘 문제 풀이 사이트 | [baekjoon.com](https://www.acmicpc.net/) |
| 💻 **프로그래머스** | 실전 코딩테스트 대비 플랫폼 | [programmers.co.kr](https://programmers.co.kr/) |
| 🏆 **LeetCode** | 글로벌 알고리즘 문제 플랫폼 | [leetcode.com](https://leetcode.com/) |
| 📊 **SWEA** | 삼성 SW Expert Academy | [swexpertacademy.com](https://swexpertacademy.com/) |

## ✨ 주요 기능

- ✅ **자동 커밋 & 푸시**: 문제를 풀면 자동으로 GitHub에 저장
- 📁 **체계적인 폴더 구조**: 플랫폼/난이도/문제번호별로 정리
- 📝 **상세한 메타데이터**: 문제 정보, 난이도, 소요 시간 등 기록
- 🔄 **실시간 동기화**: 문제 제출 즉시 레포지토리 업데이트
- 📊 **통계 관리**: README 자동 업데이트로 풀이 현황 한눈에 파악

## 🛠️ 설치 방법

### 1. 크롬 확장 프로그램 설치

```bash
# Chrome 웹 스토어에서 "BaekjoonHub" 검색 후 설치
# 또는 GitHub에서 직접 설치
git clone https://github.com/BaekjoonHub/BaekjoonHub.git
```

### 2. GitHub 연동

1. 백준허브 확장 프로그램 아이콘 클릭
2. GitHub 계정으로 로그인
3. 저장소 선택 또는 새로 생성
4. 권한 승인 후 연동 완료

### 3. 자동 푸시 설정

- 문제를 풀고 제출하면 자동으로 커밋 & 푸시됩니다
- 커밋 메시지는 자동으로 생성됩니다
  - 예시: `[백준] 1234번 - 문제이름 (Silver 3)`
